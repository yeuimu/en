Improve the Maple theme of Hexo Chinese

2024-08-31 2024-08-31 1 min 311 words Misc

The maple has main issus:

  1. The post catalogue will disapear, when window arrives certain width(such as on phon screen).
  2. The metadata will overflow over window, when window arrives certain width.

The maple uses the Tailwindcss CSS framework to write CSS, so modify its style to easy.

By order of layout.ejs->post.ejs->_partial/toc.ejs, find that toc.ejs decides post catalogue style, post.ejs decides metadata style.

Place the TOC above the title

The toc.ejs modified as:

The original style was fixed hidden lg:block, which means it only displays when the screen size exceeds the lg breakpoint. After the modification, the TOC is displayed as a block element even at smaller window widths.

Also move TOC portion in post.ejs to below section tag to ensure it is the same width as the main content and aligned with main content.

The Metadata automatically wrap

Add flex-wrap class in the section tag of the metadata.
